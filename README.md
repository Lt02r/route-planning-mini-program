# 旅游路线规划微信小程序

## 功能介绍

### 🗺️ 地图标点
- 在地图上显示所有保存的标点
- 长按地图任意位置快速添加标点
- 点击标点查看详情（名称、分类、优先级、备注、坐标）
- 支持从标点详情直接编辑或删除

### 🏷️ 标点分类显示
- 支持6种分类：景点、餐饮、住宿、购物、交通、其他
- 地图页顶部可按分类过滤显示标点
- 不同分类使用不同颜色的图钉标记

### 📋 标点增删改查
- **添加**：点击地图页右下角「＋」按钮，或长按地图，或在标点管理页添加
- **查看**：在标点管理页浏览所有标点，支持搜索和排序
- **编辑**：在标点详情或标点列表中点击进入编辑页
- **删除**：在地图详情弹窗或标点列表中删除

### 📝 自定义备注与优先级
- 每个标点支持自定义备注（最多200字）
- 支持3个优先级：普通（灰）/ 重要（橙）/ 必去（红）
- 在所有列表视图中均可看到优先级标识

### 🚗 路线规划
- 在「路线」页勾选想要前往的标点
- 支持按选择顺序规划路线
- **智能优化**：使用最近邻算法自动优化路线顺序，减少总路程
- 显示路线地图、总距离、预计出行时间
- 支持驾车/步行两种出行方式

### 🤖 AI推荐路线
- 根据可用游玩时间（1-24小时）和每个景点平均游玩时间自动计算可游玩数量
- 按优先级自动筛选最值得去的标点
- 支持按分类偏好过滤候选标点
- 自动生成优化路线，以时间轴形式展示
- 可切换到路线规划页查看完整路线

## 技术实现

- **框架**：微信小程序原生框架
- **数据存储**：wx.setStorageSync / wx.getStorageSync 本地持久化
- **路线优化**：最近邻贪心算法（Nearest Neighbor Heuristic）
- **距离计算**：Haversine 公式（球面三角法）
- **地图**：微信原生 map 组件，支持标注、折线、点击交互

```
├── utils/
│   ├── storage.js      # 本地存储工具函数
│   └── route.js        # 路线计算工具函数
└── assets/
    └── icons/          # 图标资源
```

## 开发环境

1. 下载安装 微信开发者工具
2. 打开项目目录
3. 在 project.config.json 中替换 appid 为你的小程序 AppID
4. 即可在模拟器中预览运行
